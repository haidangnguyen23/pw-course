# Kiến thức được học trong buổi 4
## 1. JavaScript - câu điều kiện
- Kiểm tra 1 đoạn logic trước khi chạy => Nếu **điều kiện đúng** mới chạy
- Các loại câu điều kiện
    - if
    - if...else
    - if...else if...else
    - switch...case
- Phổ biến nhất là Distributed Version Control System (**DVCS**)

## 2. JavaScript - vòng lặp
- Dùng để lặp lại 1 đoạn logic
- Lặp một số lần nhất định, hoặc lặp vô hạn, tuỳ theo điều kiện dừng
- Các loại vòng lặp
    - for (i)
    - for (of)
    - for (each)
    - for (in)
    - while
    - do...while
- Cú pháp vòng lặp for (i):
for (<điều kiện khởi tạo>; <điều kiện lặp>; <cập nhật>) {
// code
}
- Trong đó:
    - Điều kiện khởi tạo: chạy 1 lần duy nhất - khi vòng lặp bắt đầu
    - Điều kiện lặp: đúng thì chạy tiếp - sai thì dừng
    - Cập nhật: chạy vào mỗi cuối vòng lặp => thay đổi giá trị của biến đếm

## 3. JavaScript - Convention
- Convention (quy tắc) giúp:
    - Code theo format chung
    - Trong team dễ đọc code
- Các Convention phổ biến:
    - snake_case: tất cả viết thường, cách nhau bởi dấu gạch dưới.
    - kebab-case: tất cả viết thường, cách nhau bởi dấu gạch ngang.
    - camelCase: chữ đầu viết thường, các chữ sau viết hoa chữ cái đầu tiên.
    - PascalCase: tất cả các chữ cái đầu viết hoa.
